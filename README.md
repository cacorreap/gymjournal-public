# GymJournal — descargas

Página pública y APK de **GymJournal**, una app para crear y seguir rutinas de
gimnasio. Crea rutinas desde un video de YouTube, una foto de tu plan, texto o
pidiéndoselo a la IA. 1324 ejercicios con demostración en video, todo guardado
localmente en tu teléfono.

- **Descargar APK:** https://github.com/cacorreap/personal-projects-public-apk/releases/download/gymjournal-v1.4.0/gymjournal.apk
- **Landing:** https://cacorreap.github.io/gymjournal-public/

## Privacidad

Todos tus datos (rutinas, entrenamientos, progreso) se guardan **solo en tu
dispositivo**. La IA es opcional: usa tu propia clave de OpenAI (BYOK), que se
guarda cifrada en el teléfono. Sin clave, la app funciona en modo demo con el
catálogo local.

## Créditos
